# 领域包 · 品牌体系 / VI

## 推荐 Level

L3-L4。品牌守则改错了会影响大量物料,通常是 L4。

## 启动问用户

1. 是全新品牌还是重塑？
2. 品牌服务于什么业务（B 端 / C 端 / 奢侈品 / 快消）
3. 有多少应用场景（数字 / 实物 / 空间 / 包装）

## 常见输入分拣

| 输入 | 处置 |
|---|---|
| 设计师提议的新元素 | inbox/proposals/ |
| 应用案例需审核 | inbox/review/ → 加进 gallery/ 或拒绝 |
| 违规使用案例 | inbox/violations/ → 更新 DON'T 示例 |
| 品牌升级 / 元素修订 | 建 ADR |

## 典型 ADR

- ADR-0001：品牌定位
- ADR-0002：色系调整
- ADR-0003：logo 迭代
- ADR-0004：tone of voice 修订
